Getting Started
Follow these steps to connect your Freighter wallet and interact with BountyGuard on Stellar Testnet.
01
Install Freighter
Install the Freighter browser extension. Create or import a wallet. Make sure you are on Testnet in Freighter settings.
02
Fund Your Testnet Account
Visit friendbot.stellar.org and paste your Freighter public key to receive 10,000 XLM on testnet.
03
Add USDC Trustline
In Freighter, go to Settings → Add Asset and add USDC:

Issuer: GBBD47IF6LWK7P7MDEVSCWR7DPUWV3NY3DTQEVFL4NAT4AQH3ZLLFLA5
Code: USDC

This trustline is required to receive USDC bounty payouts.
04
Connect Wallet
Click the Connect Freighter button in the top-right. Freighter will ask you to approve the connection. Once connected, your XLM balance shows in the header and updates after every transaction.
05
Create or Verify a Bounty
Use Company View to lock USDC in escrow against a vulnerability report hash. Use Researcher View to verify a hash and check your payout status.

Deploy Guide
Step-by-step commands to build, test, and deploy BountyGuard to Stellar Testnet.
01
Install Prerequisites
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh rustup target add wasm32-unknown-unknown cargo install --locked stellar-cli --features opt stellar --version
02
Build
cd contract stellar contract build # → target/wasm32-unknown-unknown/release/bountyguard.wasm
03
Test
cargo test # 5 tests: happy path, duplicate, state, unauthorized, cancel
04
Deploy to Testnet
stellar keys generate deployer --network testnet stellar keys fund deployer --network testnet stellar contract deploy \ --wasm target/wasm32-unknown-unknown/release/bountyguard.wasm \ --source deployer \ --network testnet # → CONTRACT_ID printed here — save it
05
Initialize
stellar contract invoke --id $CONTRACT_ID \ --source deployer --network testnet \ -- initialize --admin $YOUR_FREIGHTER_ADDRESS
06
Create & Approve a Bounty
stellar contract invoke --id $CONTRACT_ID \ --source company --network testnet \ -- create_bounty \ --company $COMPANY_ADDR --researcher $RESEARCHER_ADDR \ --report_hash "sha256:abc123..." \ --usdc_amount 50000000 --usdc_token $USDC_CONTRACT stellar contract invoke --id $CONTRACT_ID \ --source company --network testnet \ -- approve_bounty \ --company $COMPANY_ADDR --report_hash "sha256:abc123..."
